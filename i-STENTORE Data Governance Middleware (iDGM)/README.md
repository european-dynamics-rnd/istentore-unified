# i-STENTORE Data Governance Middleware (iDGM)

The **i-STENTORE Data Governance Middleware (iDGM)** is the interoperability layer of the i-STENTORE digital platform. It enables secure, semantic and standardized data exchange between heterogeneous demonstrators and the platform services, ensuring interoperability across diverse energy assets, storage technologies and digital applications.

## Building Blocks

The iDGM is composed of four core building blocks:

* **iBroker** – Provides secure API management, identity and access control, request routing, and NGSI-LD context management for real-time data exchange.
* **iOntology** – Defines the common semantic model of the i-STENTORE ecosystem, extending **SAREF** and **SAREF4ENER** to represent energy assets, storage technologies, flexibility services and operational data in a standardized way.
* **iConnector** – Connects heterogeneous demonstrator systems to the i-STENTORE platform by acquiring, transforming and semantically mapping local data into the common information model.
* **iDS** – Implements the data space layer, enabling trusted, secure and sovereign data sharing between participants in accordance with European Data Spaces principles.

Together, these components provide a complete middleware solution for secure connectivity, semantic interoperability, standardized data exchange and trusted data sharing across the i-STENTORE ecosystem.
